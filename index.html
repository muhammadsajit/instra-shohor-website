Javascript Working Process

Javascript works in browser by the V8 engine.From JS source code,browser originate a parse tree.We also know that JS is an interpreted language that means it executes line by line code and show the output.JS V8 engine executes code very quickly through Just-in-time compiler. Just-in-time compiler optimize the JS code.Then it generates bytecode and translate the bytecodes into machine language.

LocalStorage VS SessionStorage

In local storage,data is exists when the browser is closed but in session storage data is not exists when browser closed.We can easily access the local storage from any windows but session storage only can access from same tab.In local storage,storage capacity is high than session storage.In local storage,users don't need to remember the information because it already exists in it but users in session storage must remember the information.Local storage not a better choice over session storage because it doesn't clear data from window for these reason any hacker can hack important information but session storage is better choice over local storage.
